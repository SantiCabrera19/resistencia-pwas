# Santi Soluciones - Plataforma Core & Demos B2B

## 🚀 Visión del Proyecto

Este repositorio alberga el núcleo digital de **Santi Soluciones**, una agencia de desarrollo de software enfocada en resolver problemas operativos reales de negocios y PyMEs (especialmente en el mercado local). 

Nuestra visión es transformar la manera en que operan las empresas, ofreciendo sistemas modernos, rápidos y centrados 100% en la Experiencia de Usuario (UX) a través de los principios de diseño Soft UI y Bento Box. No vendemos simples páginas web; vendemos velocidad en mostrador, eliminación de cuellos de botella y control absoluto de la información.

Para demostrar esta capacidad de forma tangible durante el proceso de venta, la plataforma incluye (además de la Landing Page corporativa) un ecosistema de **Prototipos Interactivos de Alta Fidelidad (Demos)** diseñados para nichos específicos:

- **Dr. Smile:** Backoffice de gestión para consultorios médicos y odontológicos. Incluye agenda reactiva, historias clínicas, facturación y configuración.
- **FerreMax:** ERP y Punto de Venta (POS) para ferreterías, corralones e industrias. Incluye carrito en caliente, control de inventario con alertas críticas de stock, CRM de clientes y tablero logístico Kanban.

## 💻 Tech Stack

El proyecto está construido bajo una arquitectura moderna y escalable, priorizando la velocidad de carga y la fluidez en las interacciones:

- **Framework Core:** [Next.js](https://nextjs.org/) (App Router)
- **Librería UI:** React 18+
- **Estilos y Maquetado:** Tailwind CSS v4
- **Micro-interacciones:** Framer Motion
- **Iconografía:** Lucide React
- **Theming:** Next-Themes (Soporte nativo Dark/Light mode libre de FOUC)

## 🛠️ Instalación y Desarrollo Local

Para correr este proyecto en un entorno local:

1. Clonar el repositorio.
2. Instalar las dependencias usando tu gestor de paquetes favorito:
   ```bash
   npm install
   ```
3. Levantar el servidor de desarrollo:
   ```bash
   npm run dev
   ```
4. Abrir `http://localhost:3000` en tu navegador.
